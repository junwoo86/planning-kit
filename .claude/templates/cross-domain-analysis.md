# 교차 도메인 분석 보고서

대상: <domain-a(확정), domain-b(검토), domain-c(컨셉)…>  ·  as-of: <YYYY-MM-DD>

## 0. 성숙도 분류 (확정/검토/컨셉)
| 도메인 | status | 역할 |
|---|---|---|
| <domain-a> | 확정 | 고정 제약 — 다른 도메인이 맞춤 |
| <domain-b> | 검토 | 검토 + 열린 결정엔 방향 제안 |
| <domain-c> | 컨셉 | 영향 판단 + 적극 방향 수정 제안 |
