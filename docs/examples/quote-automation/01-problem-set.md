# 동결 문제집합 — quote-automation

> S2에서 5whys로 근본원인까지 내려 **동결**. 이 집합은 이후 변경요청 프로토콜로만 바꾼다.

## 5whys (요약)
1. 견적 단가가 영업마다 다르다 → 왜? 단가·할인 규칙이 각자 파일/기억에 있다.
2. 왜 각자 갖고 있나 → 규칙을 모은 단일 출처가 없다.
3. 왜 단일 출처가 없나 → ERP 단가 마스터는 단가만, 할인 한도·예외 규칙은 어디에도 코드화 안 됨.
4. 왜 코드화 안 됐나 → 규칙이 영업 협상 과정에 암묵적으로만 존재.
5. **근본원인**: 견적 규칙(단가표 + 할인 승인 한도 + 예외)이 **단일 출처로 코드화돼 있지 않아** 매 견적마다 사람이 재구성 → 일관성과 속도가 동시에 무너진다. [근본원인추론] [추정]
