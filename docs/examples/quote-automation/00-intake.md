# 인테이크 — quote-automation

## 입력 분류
- 입력유형: **(b) 문제+방향** — 영업팀 진술(문제) + 대표 메모(방향) [입력] [확실]
- tier: **T1** — 부서 단위 업무 변경, 경계(결제·PII) 약함, 비가역 외부발행 없음 [AI보강] [추정]
- 참조맵 상태: 최신(이번 세션에 cross-domain 갱신) [입력]

## 맥락 요약 (S0 원본 → 정리)
- 주체: B2B 제조 부품 영업 20명, IT 인력 소수.
- 현행: 견적을 영업사원별 Excel로 작성. 단가표·할인 한도가 단일 출처 없이 흩어짐.
- 통증: 단가 불일치 / 견적 리드타임 증가 / 할인 한도 초과로 마진 훼손. [입력] [확실]
- 제약: ERP 단가 마스터는 읽기 연동만(수정 불가). 신규 대형 시스템보다 기존 흐름 삽입 선호. [입력] [확실]

## 동결 후보 문제 (S2로 전달)
1. 견적 규칙(단가·할인 한도)이 단일 출처로 코드화돼 있지 않다 → 매번 사람이 재구성. [근본원인추론] [추정]
2. 그 결과: 단가 불일치 · 리드타임 증가 · 마진 훼손(증상). [입력] [확실]
