# 입력 맥락 — 견적 자동화 (quote-automation)
<!-- 입력유형: (b) 문제+방향. 출처: 영업팀 리드 인터뷰 요약 + 대표 메모. (완주 샘플용 가상 사례) -->

## 배경
B2B 제조 부품 영업. 견적을 영업사원이 각자 Excel로 작성한다.
단가표와 할인 승인 한도가 한 곳에 정리돼 있지 않고, 영업사원 메일·개인 폴더에 흩어져 있다.

## 문제 (영업팀 진술)
- 같은 고객·품목인데 영업사원마다 단가가 다르게 나간다.
- 견적 한 건 내는 데 검토·재작성이 반복돼 시간이 오래 걸린다.
- 가끔 할인 한도를 넘겨 마진이 깎인 채 계약된다.

## 방향 (대표 메모)
"견적 규칙을 한 곳에 모으고, 영업이 그걸로 견적을 만들게 하자. 승인 한도도 자동으로 걸리게."

## 제약
- 기존 ERP(단가 마스터)는 수정이 어렵다 — 읽기 연동만 가능.
- 영업 20명, IT 인력 적음. 큰 신규 시스템보다 기존 업무 흐름에 끼우는 쪽을 선호.
