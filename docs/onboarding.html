planning-kit 설치 온보딩

개발 경험이 없어도 따라 할 수 있게 만든 설치 안내입니다. 본인 OS 탭을 고르세요.

Windows 설치 (6단계)

  1. ① 에디터 설치 — VS Code 또는 Cursor 중 하나를 설치합니다.
  2. ② Claude Code 설치 — 에디터의 확장(Extensions)에서 “Claude Code”를 설치하거나, claude.com/claude-code 안내를 따릅니다.
  3. ③ Git for Windows 설치 — git-scm.com. (이 킷의 훅·도구는 sh로 실행돼 Git Bash가 필요합니다.)
  4. ④ Python 설치 — python.org 설치 시 “Add python.exe to PATH” 체크. 설치되면 py 또는 python 명령이 생깁니다. (런처가 python3→python→py -3 순서로 자동 탐지합니다.)
  5. ⑤ (목업 쓸 때만) Node/npm + Playwright — nodejs.org 설치 후, 목업 단계(S9)를 쓸 때 Playwright MCP를 연결합니다. 보통 기획팀은 Figma를 쓰므로 선택입니다.
  6. ⑥ Notion 연동(선택) — Notion 링크 참조(*.refs.md)를 쓸 때만. 채팅에 /mcp → Notion 선택 → 브라우저 인증.

설치 성공 확인 (복사·붙여넣기)

에디터에서 이 프로젝트 폴더를 연 뒤, 터미널(Git Bash)에서:

sh .claude/hooks/launch.sh .claude/tools/completion-gate.py docs/examples/quote-automation --mode report
예상 출력: 마지막 줄에 판정: 보고서 생성 가능 … 이 나오고 종료코드가 2면 성공입니다(완주 샘플 quote-automation 기준). 한글이 깨지면 런처가 UTF-8을 고정하므로 보통 정상 표시됩니다.

macOS 설치 (6단계)

  1. ① 에디터 설치 — VS Code 또는 Cursor 중 하나를 설치합니다.
  2. ② Claude Code 설치 — 에디터 확장에서 “Claude Code” 설치 또는 claude.com/claude-code 안내.
  3. ③ 터미널·sh — macOS는 기본 sh/zsh가 있어 별도 설치 불필요. (Xcode Command Line Tools가 없으면 xcode-select --install.)
  4. ④ Python 설치 — python.org 또는 Homebrew brew install python. python3 명령이 생깁니다. (런처가 자동 탐지.)
  5. ⑤ (목업 쓸 때만) Node/npm + Playwright — brew install node 후 Playwright MCP 연결. 기본 경로는 Figma라 선택입니다.
  6. ⑥ Notion 연동(선택) — 채팅에 /mcp → Notion 선택 → 브라우저 인증.

설치 성공 확인 (복사·붙여넣기)

에디터에서 이 프로젝트 폴더를 연 뒤, 터미널에서:

sh .claude/hooks/launch.sh .claude/tools/completion-gate.py docs/examples/quote-automation --mode report
예상 출력: 마지막 줄에 판정: 보고서 생성 가능 …, 종료코드 2면 성공입니다(완주 샘플 기준).
