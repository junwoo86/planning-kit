# docs/planning/ — 진행 산출물 (로컬 전용)

`/strategy-plan`이 기획을 진행하며 **`docs/planning/<slug>/`** 아래에 산출물을 남기는 작업 디렉토리입니다
(`_state.md` 재개 앵커 + `00-intake.md` … `11-handoff.md` + `REPORT.md`).

> ⚠️ **이 폴더의 `<slug>/` 내용은 레포에 올라가지 않습니다**(`.gitignore`). 사용자 컨텍스트·산출물은
> **각자 로컬에서만** 관리합니다. 레포는 *뼈대*만 공유합니다.

**완성된 산출물의 모습이 궁금하면** → 동봉 예시 [`../examples/quote-automation/`](../examples/quote-automation/)
(제로베이스로 시작하려면 그냥 `/strategy-plan`을 실행하세요 — 이 폴더는 비어 있는 게 정상입니다).
