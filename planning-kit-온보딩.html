무엇을 까나요

에디터 + Claude Code

코드 에디터(VS Code/Cursor) 안에서 Claude Code(AI 비서)가 돌아갑니다. 여기에 킷 폴더를 열어 채팅으로 일합니다.

왜 필요한가요

킷이 도구를 직접 돌립니다

킷은 점검·게이트 같은 작은 프로그램을 자동 실행해요. 그래서 Python과 (Windows는) Git Bash가 필요합니다.

2단계별 설치 — 클릭만 따라오세요

본인 운영체제 탭을 고르세요. 각 단계의 파란 링크를 누르면 공식 다운로드 페이지로 갑니다.

코드 에디터를 설치합니다 — VS Code(추천) 또는 Cursor 중 하나.
다운로드 → 실행 → 기본값으로 설치. 설치 후 한 번 열어 두세요.
Claude Code를 연결합니다 — 에디터 왼쪽 확장(Extensions) 아이콘 → “Claude Code” 검색 → 설치. (안내: claude.com/claude-code)
설치 후 안내대로 로그인까지 마치면 에디터 안에 채팅창이 생깁니다 — 여기서 킷과 대화합니다.
Git for Windows를 설치합니다 — git-scm.com/download/win.
‘다음 → 다음’ 기본값으로 설치하면 됩니다. (킷의 점검 도구가 Git Bash로 돌아가고, 킷을 내려받을 때도 씁니다.)
Python을 설치합니다 — python.org에서 내려받아 실행.
⚠️ 설치 첫 화면 맨 아래 “Add python.exe to PATH” 체크박스를 꼭 켜고 설치하세요. (킷이 py/python을 자동으로 찾습니다.)
킷을 내려받습니다 — 둘 중 편한 방법으로:
쉬운 길: github.com/junwoo86/planning-kit → 초록 Code 버튼 → Download ZIP → 압축 풀기.
또는 Git Bash에서: git clone https://github.com/junwoo86/planning-kit.git
→ 받은 planning-kit 폴더를 에디터에서 ‘폴더 열기(Open Folder)’로 엽니다.

코드 에디터를 설치합니다 — VS Code(추천) 또는 Cursor 중 하나.

3설치가 잘 됐는지 1줄로 확인

에디터에서 planning-kit 폴더를 연 상태에서, 터미널(Windows는 Git Bash)에 아래 한 줄을 복사·붙여넣기 하세요.

sh .claude/hooks/launch.sh .claude/tools/completion-gate.py docs/examples/quote-automation --mode report
✅ 이렇게 나오면 성공 — 마지막 줄에 판정: 보고서 생성 가능 … 이 보이면 됩니다(동봉된 예시 quote-automation으로 점검 도구가 정상 작동한 것). 한글이 깨져 보여도 걱정 마세요 — 실제 동작엔 문제 없습니다.
⚠️ 잘 안 되면 — 가장 흔한 원인은 ⑴ 에디터에서 planning-kit 폴더를 안 연 상태(다른 폴더에서 실행), ⑵ Python 미설치/‘PATH 체크’ 누락, ⑶ Windows에서 일반 명령창(cmd)이 아니라 Git Bash로 실행해야 함. 그래도 막히면, 채팅에 “설치 확인이 안 돼”라고 하면 킷이 무엇이 빠졌는지 같이 짚어 줍니다.

4Notion 연동 선택

팀의 Notion 문서를 참고 자료로 연결할 때만 하면 됩니다. 안 써도 킷은 정상 작동해요.

연결(한 번만) — 에디터 채팅에 /mcp 입력 → 목록에서 “claude.ai Notion” 선택 → 브라우저에서 로그인.
그 뒤로는 연결된 계정에 공유된 페이지만 읽어 참고합니다.
링크를 적습니다 — .refs.md 파일에 Notion URL을 한 줄에 하나씩. 자세한 방법·위치는 사용설명서 §4 Notion 참조.
🔒 안전 규칙 — 인증이 안 됐거나 권한이 없으면 킷은 내용을 지어내지 않고 [확인불가]로 정직하게 남깁니다(조용히 넘어가지 않음). 또 finalized/로 연결한 Notion 문서는 읽기 전용(킷이 절대 수정하지 않음)입니다.
